This is a second static HTML page with a different design. You can upload it to your new S3 bucket created using the AWS CLI.

Use it to test that your bucket works correctly once deployed.

AWS CLI Instructions

How to Create a Bucket via AWS CLI

aws s3api create-bucket \
  --bucket my-cli-created-bucket \
  --region eu-central-1 \
  --create-bucket-configuration LocationConstraint=eu-central-1

Then upload the files:

aws s3 cp index.html s3://my-cli-created-bucket/
aws s3 cp styles.css s3://my-cli-created-bucket/

Enable static website hosting via the AWS Console or CloudFormation.
